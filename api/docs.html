Chase Deux API Documentation

GET /obstacleLimit

Returns a number that represents the number of times Jerry can hit an obstacle before Tom catches up. 

Example Response:
{
    "obstacleLimit": 3
}

GET /image

Returns an image of the requested character.

Query Parameter:

  • character (string) - can take values "tom", "jerry", "obstacle". 
Example Request:
GET /image?character=tom

Example Response:
Image of tom in .png format

GET /hitHindrance

Returns a random reward/punishment to be given, on hitting a type of obstacle in hacker mode. 

Possible Effects:
- Type 1 : Increase speed of run by given amount
- Type 2 : Auto jump upcoming obstacles for given seconds
- Type 3 : Bring Tom closer by given threshold and if he crosses Jerry, let him catch Jerry

Example Response:
{
    "type":2,
    "amount":4,
    "description":"Auto jump upcoming obstacles for given seconds"
}
    Meaning : Let Jerry automatically evade and escape the upcoming obstacles for the next 4 (since "amount" : 4 in the response) seconds after hitting the hindrance.

GET /obstacleCourse

Returns a character array of a specified length depicting the obstacle course. The characters "L", "M", "R", "B" represent different types of obstacles. 

Example Response:
{
    "obstacleCourse": ["L", "M", "R", "B", "L", "M", "R", "B"]
}

POST /randomWord

Generates a random word of a specified length (between 5 and 15 characters) and returns it.

Request Parameter:

  • length (int) - the length of the word.
Example Request Body:
POST /randomWord
{
    "length": 7
}

Example Response:
{
    "word": "android"
}
